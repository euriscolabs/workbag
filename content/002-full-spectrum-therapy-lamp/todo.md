# Video Todo — Full-Spectrum Therapy Lamp

## Pre-production

- [ ] Clarify goal + audience
- [ ] Choose title direction + 2-5 backups
- [ ] Thumbnail concepts (at least 3)
- [ ] Research / references collected
- [ ] Script bullets (or full script)
- [ ] Storyboard drafted
- [ ] Shot list and required assets ready

## Production

- [ ] Record A-roll
- [ ] Record B-roll / screen capture
- [ ] Collect music/SFX (if needed)
- [ ] Back up footage

## Post-production

- [ ] Rough cut
- [ ] Tighten pacing / remove dead time
- [ ] Add text/graphics
- [ ] Audio cleanup
- [ ] Color / grading (if needed)
- [ ] Export and review
- [ ] Identify 3-10 short-form clip moments (hooks + payoffs)
- [ ] Cut/format shorts for TikTok / Reels / YouTube Shorts (captions, safe zones)

## Publish

- [ ] Final title
- [ ] Final thumbnail selected
- [ ] Description written + key links
- [ ] Chapters (if long-form)
- [ ] Tags filled
- [ ] End screens / cards set
- [ ] Schedule / publish
- [ ] Short-form posting plan set (TikTok / Instagram / Shorts)

## After publish

- [ ] First-hour engagement plan
- [ ] Share to relevant places
- [ ] Review retention graph + notes
- [ ] Post 1-3 shorts/reels/tiktoks as entry points back to the channel
