# Storyboard — Biofilter for micro/nanoplastics

| Time | Beat | Visual | Audio | Notes |
|------|------|--------|-------|------|
| 0:00-0:10 | Hook | Filter reveal + "microplastics?" text | Strong cold open | Set the question |
| 0:10-0:30 | Promise | "Build + test + what I learned" | Voiceover | Keep it simple |

| 1:10-4:00 | Build | Parts + assembly + flow test | On-cam + VO | Show layers clearly |
