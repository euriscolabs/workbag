## Overview

A fitness application (originally "Lift-Pal") that allows coaches to create, distribute, and track personalized workout plans for their athletes. Athletes get instant access to their plans and progress tracking, while built-in AI suggests exercises, loads, and schedules.

## Problem

Coaches and athletes rely on Excel sheets, complicated websites, or fragmented free apps to manage workout plans. Existing fitness apps lack comprehensive features, coach-athlete workflows, or AI-powered recommendations — and the good ones are expensive.

## Approach

### Business Model

1. Coach subscribes to become a "coach" user
2. Coach creates workout plans, each with a unique access code
3. Athletes pay the coach directly for a plan
4. Coach gives the workout code to the athlete
5. Athlete uses the code to create their login and access plans

### Key Features

- Workout plan creation and distribution (coach → athlete)
- Progress tracking and live stats visibility for coaches
- AI-powered workout recommendations (exercises, loads, schedules)
- Social features: profiles, follow, achievements, PRs, highlights
- Time-based circuit/flow support (yoga, core, hybrid)
- Exercise database (gym, cardio, custom)

### Competitive Edge

Based on competitor analysis (GymKeeper, FitHero, StrongLifts 5x5), must-haves:
- Simple, clean interface
- Workout database (gym, cardio, custom)
- History view
- Minimal ads / fair pricing
- Forecast and recommendation features (AI)
