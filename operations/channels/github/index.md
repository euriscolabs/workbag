# GitHub

## Organization
- Org: euriscolabs
- URL: https://github.com/euriscolabs

## Purpose
Open source home for all research, code, CAD files, and schematics.

## Repository Strategy
| Repo type | Naming convention | License |
|-----------|------------------|---------|
| Project repos | `project-name` | CERN-OHL-S (hardware) / MIT (software) |
| Research data | `research-topic` | CC-BY |
| Website | `website` | MIT |
| Tooling | `tool-name` | MIT |

## Workflows
- CI/CD for website deployment (already in place)
- Automated releases for software projects
- Issue templates for bug reports and feature requests

## Best Practices
- Consistent README structure across repos
- Clear contribution guidelines
- Proper licensing in every repo
- Release tags for stable versions
- Project boards for tracking progress
