# Channels

All platforms where Eurisco Labs publishes content and engages with the community.

| Channel | Primary Purpose | Status |
|---------|----------------|--------|
| [YouTube](youtube/) | Long-form video, tutorials, build docs | Active |
| [Instagram](instagram/) | Visual content, behind-the-scenes | Active |
| [TikTok](tiktok/) | Short clips, viral reach | Active |
| [GitHub](github/) | Open source, research data, code | Active |
| [Patreon](patreon/) | Community, recurring revenue | Setup pending |
| [Newsletter](newsletter/) | Direct audience, updates | Planned |

## Cross-Posting Strategy

1. **Long-form video** (YouTube) — the anchor content
2. **Clips** (TikTok, Instagram Reels, YT Shorts) — extracted highlights
3. **Behind-the-scenes** (Instagram Stories, Patreon) — process shots, fails, extras
4. **Written summary** (Newsletter) — recap + links + exclusive insights
5. **Open data** (GitHub) — code, CAD files, research data
6. **Extras** (Patreon) — print-ready files, guides, early access
