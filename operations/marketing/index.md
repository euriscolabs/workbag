# Marketing

Strategy and execution for growing Eurisco Labs reach and revenue.

## Areas

| Area | Purpose | Status |
|------|---------|--------|
| [SEO](seo/) | Organic discovery across web and YouTube | Active focus |
| [Ads](ads/) | Paid promotion for products and content | Planned |
| [Content Calendar](content-calendar/) | Unified schedule across all channels | To set up |

## Funnel

```
Discovery (SEO, Ads, Shorts/TikTok)
    → YouTube (long-form, trust-building)
        → Website (projects, credibility)
            → Webshop (purchase) / Patreon (subscribe)
                → Newsletter (retention, repeat)
```

## Key Metrics to Track
- YouTube: views, watch time, subscribers, CTR
- Website: visitors, bounce rate, time on site
- Webshop: conversion rate, average order value
- Patreon: subscriber count, MRR, churn
- Newsletter: open rate, click rate, growth
- Social: followers, engagement rate, reach
