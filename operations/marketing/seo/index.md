# SEO

## Strategy
Optimize for organic discovery on both Google and YouTube. Every piece of content should be findable by people searching for the problem it solves.

## Website SEO

### Technical
- [ ] Fast page loads (static site helps)
- [ ] Mobile-friendly responsive design
- [ ] Proper meta tags (title, description, og:image)
- [ ] Structured data / schema markup (Product, Article, HowTo)
- [ ] Sitemap.xml
- [ ] robots.txt
- [ ] Canonical URLs
- [ ] Internal linking between related projects

### Content
- [ ] Each project page targets a specific search query
- [ ] Long-tail keywords (e.g., "DIY nanoplastic water filter" vs "water filter")
- [ ] Blog posts around research topics (educational, linkable)
- [ ] Alt text on all images
- [ ] Descriptive URLs (e.g., /projects/biofilter-nanoplastics)

### Off-Page
- [ ] Backlinks from maker/science communities
- [ ] GitHub repos linking back to website
- [ ] YouTube descriptions linking to project pages
- [ ] Forum participation (Reddit, Hacker News, maker forums)

## YouTube SEO

### Per Video
- [ ] Keyword research before scripting
- [ ] Keyword in title (front-loaded)
- [ ] Descriptive title (searchable, not clickbait-only)
- [ ] First 2 lines of description contain target keyword
- [ ] Full description with timestamps, links, keywords
- [ ] Tags (broad + specific)
- [ ] Custom thumbnail (high contrast, readable text)
- [ ] End screens + cards

### Channel
- [ ] Channel keywords set
- [ ] Channel description optimized
- [ ] Playlists organized by topic
- [ ] Consistent upload schedule (algorithm favors consistency)
