# Ads

## Status
Planned — start once there are products to sell and a baseline organic audience.

## Platforms

| Platform | Best For | Ad Type |
|----------|----------|---------|
| Google Ads | Product searches, high intent | Search, Shopping |
| YouTube Ads | Awareness, retargeting | Pre-roll, discovery |
| Instagram/Facebook Ads | Visual products, retargeting | Reels, Stories, Feed |
| TikTok Ads | Viral reach, younger audience | In-feed, Spark Ads |

## Strategy

### Phase 1: Foundation (before spending)
- [ ] Set up Google Analytics / tracking pixels on website and webshop
- [ ] Define target audience personas
- [ ] Identify top-performing organic content (ads should amplify what already works)

### Phase 2: Testing (small budget)
- [ ] Start with retargeting (people who visited site but didn't buy)
- [ ] Boost best-performing YouTube videos
- [ ] Test product ads on Google Shopping
- [ ] A/B test ad creatives and copy
- [ ] Set daily budget caps to control spend

### Phase 3: Scaling (once ROAS is positive)
- [ ] Increase budget on winning campaigns
- [ ] Expand to lookalike audiences
- [ ] Seasonal campaigns (holidays, back to school, etc.)
- [ ] Sponsorship-style YouTube collaborations
