## Overview

A parametric, 3D-printable tool inlay system that lets you create custom-fit tool holders for drawers, pegboards, or workbenches — designed to be easily configurable for any tool.

## Problem

Off-the-shelf tool organizers rarely fit your specific tools or workspace. Custom foam inlays work but aren't durable, reusable, or easy to modify.

## Approach

- Parametric design (OpenSCAD or FreeCAD) — adjust dimensions per tool
- Modular grid system that tiles together
- Print on the Prusa XL for large inlays in one piece

## Open Questions

- Parametric tool: OpenSCAD vs FreeCAD vs Fusion 360?
- Base grid dimensions and snap-fit mechanism
- How to handle odd-shaped tools (custom profiles vs. generic slots?)
- Wall-mount vs. drawer vs. both?
